# Triage Labels

| Role | GitHub label |
|---|---|
| Needs initial assessment | `needs-triage` |
| Needs reporter information | `needs-info` |
| Ready for an implementation agent | `ready-for-agent` |
| Requires a human decision or action | `ready-for-human` |
| Intentionally declined | `wontfix` |

Defects additionally carry `bug`.
